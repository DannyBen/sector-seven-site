Overview

Tiny Speech is a portable system tray utility for Windows that adds a global hotkey for speaking any selected text using the default (SAPI) Windows Text-to-Speech.

Usage

  • F1 - Speak selected text
  • F3 - Pause/Resume speaking
  • F4 - Stop speaking

Configuration

Running TinySpeech for the first time will create a TinySpeech.ini file in the same directory. This file can be used to configure hotkeys.
