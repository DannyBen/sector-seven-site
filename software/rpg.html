RPG - Renoise Phrase Generator

Melody Generator for Renoise

Version
0.8
Last Release
May 10, 2014

Overview

RPG is a utility that generates random tracks for Renoise, based on a user defined set of parameters.

RPG is working directly inside Renoise - while it is running, you may press a hotkey to generate new tracks that are pasted into Renoise.
